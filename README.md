# Conversor de Imagens `.webp` para `.jpg`

Este projeto é um simples conversor de imagens que transforma arquivos no formato `.webp` para `.jpg`. Ele utiliza a biblioteca `Pillow` em Python para abrir e converter as imagens.

## Funcionalidades

- Converter arquivos de imagem do formato `.webp` para `.jpg`.
- Processar múltiplos arquivos `.webp` e salvá-los como `.jpg`.

## Pré-requisitos

- **Python** (versão 3.6 ou superior).
- **Pillow**: biblioteca para manipulação de imagens.

Para instalar a biblioteca `Pillow`, você pode usar o seguinte comando:

```bash
pip install Pillow
